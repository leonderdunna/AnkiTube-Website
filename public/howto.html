AnkiTube

Karten erstellen

Wenn die ↓ AnkiTube Erweiterung installiert ist, erscheint neben YouTube Videos ein bereich zum Erstellen von Karten. Karten die erstellt wurden, werden automatisch online gespeichert und können von allen Nutzern der Erweiterung angesehen werden.
Es gibt 2 Kartentypen die zurzeit unterstützt werden

Standard

Einfache "Frage Antwort" Karte.

Lückentext

Dem Nutzer wird der gesamte Text der Karte gezeigt, mit Ausnahme der durch {{c1:: ... }} gekennzeichneten bereiche. Dabei wird immer ein Teil weggelassen. Kennzeichnet man z.B. drei Bereiche mit c1, c2 und c3, so werden von Anki 3 Karten erstellt. eine bei der alle c1 Teile fehlen, eine bei der alle c2 Teile fehlen und eine bei der alle c3 Teile fehlen. Gekennzeichnete Bereiche dürfen sich dabei nicht überlappen.

Mathe

LaTex wird unterstützt, indem LaTex Formeln zwischen \( und \) geschrieben werden
Beispiel: um "ein halb" zu schreiben kann man in den Kartentext (sowohl Standard als auch Lückentext) "\( \frac{1}{2} \)" schreiben

Karten Herunterladen

um nur die Karten von einem Video herunterzuladen kann man in dem Feld neben dem Video den Herunterladen-Knopf drücken. Dadurch bekommt man eine Datei, die man über das Anki interne Import-System in den eigenen Kartensatz hinzufügen kann. Der ↓↑ Button aktualisiert alle zu dem Video Gehörenden karten.

Decks Erstellen

Mehrere Videos können zu Decks zusammengefasst werden. ein neues Deck kann durch die + Schaltfläche erstellt werden. Innerhalb von jedem deck kann durch "+ Deck" ein Unterdeck hinzugefügt werden und durch "+ Video" zum Entsprechenden deck ein Video. "X" Löscht das Deck, Unterdeck oder Video.

Decks Herunterladen

Über das Anki Import-System können keine neuen Decks erstellt werden, daher wird CrowdAnki genutzt.
Durch den "↓" Button kann ein Deck heruntergeladen werden. Die "deck.json" Datei muss in einem leeren Ordner gespeichert werden. In Anki kann diese (sofern CrowdAnki installiert ist) im Menü unter "Datei → CrowdAnki: Import from Disk" importiert werden.
